================================================================================
ÖNEMLİ PROMPT PATTERN'LERİ
================================================================================

 BAŞARILI PATTERN:
1. Spesifik gereksinimler ver (teknoloji stack, constraint'ler)
2. "Neden" sorusunu sor (ChatGPT: "Neden React Query overkill?")
3. Alternatifler iste (3 AI'dan aynı soruyu sor, karşılaştır)
4. Edge case'leri explicit belirt

 BAŞARISIZ PATTERN:
1. Genel sorular ("En iyi framework ne?") - context eksik
2. Versiyon belirtmeden ("Tailwind kurulumu") - breaking change riski
3. Deprecated API kullanımı (AI'nın bilgisi cut-off date'e kadar)
4. "Tam çözüm ver" (AI copy-paste yerine hibrit yaklaşım)
